# Variable-Yield USD Stablecoin (sCBD)

Curve Bonds DAO’s variable-yield stablecoin is designed to provide **passive exposure to optimized crvUSD yield strategies**.

Users deposit into a single asset and gain access to yield generated across Curve pools and LlamaLend markets, without needing to actively manage positions.

This product is represented by sCB&#x44;**+**, a yield-bearing token that increases in value over time as yield is accrued.
