# Introducing Curve Bonds DAO

Curve Bonds DAO is a decentralized protocol built on Curve that offers simplified variable-yield and fixed-rate stablecoin and ETH Derivative products.

It is designed for users who want to earn **competitive yields on stablecoins** and ETH without actively managing positions across Curve pools and LlamaLend markets.

The protocol achieves this by:

* Aggregating and deploying capital into leading Curve strategies
* Offering a **variable-yield stablecoin, a variable-yield ETH derivative** and **fixed-rate USD/ETH bond products**
* Capturing and reinvesting protocol revenue to create a **compounding treasury base that helps maintain 100%+ collateralization levels in case any deployed strategies suffer loss.**&#x20;
