# Governance Framework

Curve Bonds DAO utilizes Aragon to secure protocol contracts and treasury assets through a structured, multi-layer governance system.

Governance is divided into two distinct processes designed to balance flexibility, security, and operational efficiency:

1. DAO Upgrade Process (Full Governance)
2. Treasury Optimization Process (Operational Governance)
